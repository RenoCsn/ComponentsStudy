<h1 align="center">Esta é uma biblioteca de componentes desenvolvida para estudo e portfólio.</h1>

Este projeto usa o design do [material UI](https://mui.com/) como exemplo a ser seguido, porem pode conter algumas adaptações.

Inicialmente só está acessível por download + "npm install", para fazer a instalação do projeto e "npm run storybook" para rodar o projeto e poder interagir com os componentes que ja foram desenvolvidos.

Pretendo fazer o upload do storybook em um host publico para facilitar o acesso.
